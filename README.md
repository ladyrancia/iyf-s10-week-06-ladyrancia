# Week 6 - JavaScript Async & Fetch

## Features Implemented

### 1. Promises

* Promise creation
* Promise chaining
* Promise.all
* Promise.race

### 2. Async/Await

* Converted promise chains
* Error handling with try/catch

### 3. Fetch API

* GET requests
* POST requests

### 4. DOM Manipulation

* Display posts on webpage
* Dynamic content rendering

### 5. Extra Features (Advanced)

* Load more posts (pagination)
* Search functionality
* Delete posts
* Styled UI

## How to Run

### Node (for app.js)

```bash
node app.js
```

### Browser (for UI)

Open index.html with Live Server
